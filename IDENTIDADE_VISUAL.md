# MedUp Pagamentos - Identidade Visual

## Tema Escuro

A aplicação utiliza um tema escuro profissional com a paleta de cores oficial da MedUp.

## Paleta de Cores

### Cores Principais
- **#042938** - Azul escuro (Primary)
  - Usado em: fundo principal da página
- **#96d4ff** - Azul claro (Secondary)  
  - Usado em: botões primários, elementos de destaque, luzes
- **#ffffff** - Branco (White)
  - Usado em: texto principal, contraste

### Cores de Apoio
- **#031a24** - Azul mais escuro (Dark)
  - Usado em: cards, elementos secundários
- **#021015** - Azul mais escuro ainda (Darker)
  - Usado em: navbar, headers, elementos de destaque

## Efeitos Visuais

### Sombras e Luzes
- **Sombras escuras**: `rgba(0, 0, 0, 0.3)` para profundidade
- **Sombras coloridas**: `rgba(150, 212, 255, 0.2)` para efeitos de luz
- **Bordas luminosas**: `rgba(150, 212, 255, 0.1)` para definição sutil
- **Efeitos hover**: Glow azul claro para interatividade

### Cards e Elementos
- Fundo escuro com bordas luminosas
- Sombras múltiplas para profundidade
- Efeitos de luz interna (inset shadows)
- Transições suaves em hover

## Logo

O logo da MedUp está integrado em:
- **Favicon**: `/public/favicon.svg` (versão simplificada)
- **Navbar**: Logo + texto "MedUp Pagamentos"
- **Título da página**: "MedUp Pagamentos - Status dos Clientes"

## Elementos Visuais

### Cards de Estatísticas
- Fundo escuro com bordas luminosas
- Sombras múltiplas para profundidade
- Efeito hover com glow azul
- Transições suaves

### Botões
- **Primário**: Fundo azul claro, texto escuro
- **Outline**: Borda azul claro, hover com fundo azul claro
- Sombras coloridas em hover

### Formulários
- Fundo escuro com bordas luminosas
- Focus states com glow azul
- Placeholders em azul claro translúcido

### Tabelas
- Fundo escuro com linhas luminosas
- Hover states com brilho sutil
- Headers mais escuros para contraste

### Modais
- Fundo escuro com sombras profundas
- Bordas luminosas para definição
- Headers mais escuros

## Responsividade

A identidade visual escura é mantida em todos os tamanhos de tela:
- Mobile: Elementos adaptados mantendo contraste
- Tablet: Layout otimizado com sombras apropriadas
- Desktop: Experiência completa com todos os efeitos

## Acessibilidade

- Alto contraste entre texto branco e fundos escuros
- Estados de foco com glow azul visível
- Sombras sutis que não interferem na legibilidade
- Bordas luminosas para definição de elementos
