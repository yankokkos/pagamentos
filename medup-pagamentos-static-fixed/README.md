# MedUp Pagamentos - Versão Estática 
 
Esta é uma versão estática do MedUp Pagamentos que funciona em qualquer hospedagem web. 
 
## Como usar: 
 
1. Faça upload de todos os arquivos da pasta `dist-static` para sua hospedagem 
2. Acesse o arquivo `index.html` no seu navegador 
3. A aplicação funcionará com dados de demonstração 
 
## Arquivos incluídos: 
- `index.html` - Página principal 
- `login.html` - Página de login 
- `favicon.svg` - Ícone do site 
- `css/style.min.css` - Estilos minificados 
- `js/app-static.js` - JavaScript com dados de exemplo
